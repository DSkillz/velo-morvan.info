Choisir son vélo pour le Morvan

Le choix du vélo est crucial pour profiter pleinement du Morvan. Le relief vallonné, les chemins variés et les distances importantes nécessitent un vélo adapté à vos objectifs et à votre niveau. Ce guide vous aide à choisir entre vélo de route, VTT, VAE et gravel selon vos envies.

Vélo de route

Pour qui : cyclistes sur route recherchant performance et vitesse

Avantages

  • Rendement optimal sur routes goudronnées
  • Légèreté, vitesse en descente
  • Idéal pour les cols et les longues distances

Inconvénients

  • Limité aux routes goudronnées
  • Moins confortable sur routes abîmées
  • Pas adapté aux chemins forestiers

Recommandation

Choisissez un vélo de route si vous restez exclusivement sur les routes départementales et nationales du Morvan. Privilégiez un compact (50/34) pour les montées raides.

VTT (Vélo Tout-Terrain)

Pour qui : amateurs de chemins forestiers, single tracks et terrains techniques

Avantages

  • Polyvalence totale : routes, chemins, forêts
  • Confort grâce aux suspensions
  • Adhérence et stabilité sur tous terrains
  • Parfait pour explorer le Morvan hors des sentiers battus

Types de VTT

  • Semi-rigide : suspension avant uniquement, léger et efficace
  • Tout-suspendu : suspensions avant et arrière, confort maximal
  • Trail : polyvalent, idéal pour le Morvan (120-140 mm de débattement)

Recommandation

Le VTT est le vélo roi dans le Morvan pour accéder à tous les chemins. Un trail semi-rigide ou tout-suspendu de 120-130 mm est parfait pour les circuits du Morvan.

VAE (Vélo à Assistance Électrique)

Pour qui : tous publics souhaitant dompter les dénivelés sans effort excessif

Avantages

  • Dénivelés et distances accessibles à tous
  • Maintien d'un rythme régulier en montée
  • Idéal pour groupes de niveaux hétérogènes
  • Parfait pour seniors ou cyclistes occasionnels

Types de VAE

  • VAE route : pour routes goudronnées
  • VTT électrique : pour chemins et forêts
  • VTC électrique : polyvalent, confortable

Critères importants

  • Autonomie : minimum 400 Wh pour le Morvan (idéal : 500-600 Wh)
  • Couple moteur : 60 Nm minimum pour les montées
  • Poids : 20-25 kg, manipuler avec précaution

Recommandation

Le VAE démocratise le cyclisme dans le Morvan. Privilégiez un VTC ou VTT électrique avec batterie 500 Wh minimum pour gérer le dénivelé.

Gravel

Pour qui : cyclistes polyvalents cherchant un compromis route/chemins

Avantages

  • Polyvalence route + chemins roulants
  • Plus rapide qu'un VTT sur route
  • Confort grâce aux pneus larges
  • Léger et maniable

Limites

  • Moins performant qu'un VTT sur chemins techniques
  • Single tracks et forêts difficiles

Recommandation

Le gravel est un bon compromis si vous alternez routes secondaires et chemins roulants. Évitez les parcours trop techniques en forêt.
