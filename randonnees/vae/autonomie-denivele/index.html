Autonomie VAE et dénivelé dans le Morvan

L'autonomie d'un VAE dans le Morvan dépend fortement du dénivelé et du mode d'assistance utilisé. Comprendre ces facteurs est essentiel pour planifier vos sorties sans risquer la panne de batterie en pleine nature. Ce guide vous aide à estimer votre autonomie réelle selon le terrain.

Facteurs influençant l'autonomie

  • Dénivelé positif : le facteur le plus important dans le Morvan
  • Mode d'assistance : éco, normal, sport, turbo
  • Poids du cycliste : plus le poids est élevé, plus la consommation augmente
  • Cadence de pédalage : pédaler trop lentement consomme plus
  • Vent : vent de face réduit l'autonomie de 10-15%
  • Température : le froid réduit la capacité de la batterie
  • Pression des pneus : pneus sous-gonflés augmentent la consommation

Autonomie selon le dénivelé

Pour une batterie standard de 500 Wh :

Terrain plat (0-100 m D+)

  • Mode éco : 100-120 km
  • Mode normal : 70-90 km
  • Mode sport : 50-60 km

Relief modéré (100-400 m D+)

  • Mode éco : 70-90 km
  • Mode normal : 50-65 km
  • Mode sport : 35-45 km

Relief moyen (400-800 m D+)

  • Mode éco : 50-65 km
  • Mode normal : 35-50 km
  • Mode sport : 25-35 km

Relief important (800+ m D+)

  • Mode éco : 35-50 km
  • Mode normal : 25-35 km
  • Mode sport : 20-28 km

Optimiser son autonomie

  • Adapter le mode : utilisez l'éco en plat, normal en montée douce, sport en côte raide
  • Pédaler efficacement : maintenez une cadence de 60-80 tours/minute
  • Anticiper : passez en mode éco avant d'arriver en haut
  • Descentes : coupez l'assistance en descente pour économiser
  • Démarrage : ne démarrez pas en mode turbo
  • Entretien : chaîne bien lubrifiée = moins de consommation
  • Pneus : vérifiez la pression avant chaque sortie

Estimer l'autonomie d'un parcours

Règle pratique pour le Morvan :

Autonomie estimée = Capacité batterie (Wh) / [(Dénivelé positif en m / 100) × 12]

Exemple : Batterie 500 Wh, parcours 600 m D+ en mode normal
500 / [(600 / 100) × 12] = 500 / 72 ≈ 40-45 km d'autonomie
