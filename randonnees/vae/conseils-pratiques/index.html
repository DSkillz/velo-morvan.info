Conseils pratiques VAE dans le Morvan

Bien utiliser un VAE dans le Morvan nécessite quelques connaissances spécifiques. Ce guide pratique vous donne toutes les astuces pour profiter pleinement de votre vélo électrique, gérer la batterie efficacement et éviter les désagréments en pleine randonnée.

Bien utiliser les modes d'assistance

Mode éco (assistance faible)

  • Quand : terrain plat, descentes légères, portions faciles
  • Avantage : maximise l'autonomie
  • Effort : pédalage proche d'un vélo classique

Mode normal (assistance moyenne)

  • Quand : faux-plat, montées douces, usage polyvalent
  • Avantage : bon compromis autonomie/confort
  • Effort : effort modéré, confortable

Mode sport/turbo (assistance forte)

  • Quand : montées raides, démarrages, passages difficiles
  • Avantage : effort minimal
  • Inconvénient : consomme beaucoup de batterie

Entretien de la batterie

  • Stockage : conservez la batterie entre 30% et 60% de charge si inutilisée
  • Température : évitez les températures extrêmes (idéal : 10-25°C)
  • Recharge : rechargez après chaque sortie, même partiellement
  • Éviter : ne laissez jamais la batterie à 0% pendant plusieurs jours
  • Durée de vie : 500-800 cycles de charge (3-5 ans selon usage)
  • Nettoyage : nettoyez les contacts régulièrement avec un chiffon sec

Recharge dans le Morvan

Si vous planifiez une sortie longue :

  • Emportez le chargeur : certains restaurants/cafés acceptent de recharger pendant le repas
  • Temps de charge : 3-6h pour une charge complète selon le modèle
  • Charge partielle : 1h de charge = environ 20-30 km d'autonomie
  • Bornes : peu de bornes publiques VAE, ne comptez pas dessus
  • Hébergement : vérifiez que vous pourrez recharger sur place

Sécurité et entretien VAE

  • Freins : les VAE sont plus lourds, vérifiez les freins régulièrement
  • Pneus : contrôlez la pression avant chaque sortie (bar indiqué sur le flanc)
  • Chaîne : lubrifiez tous les 200-300 km
  • Vitesses : passez les vitesses avant les montées, pas pendant
  • Poids : VAE = 20-25 kg, soyez prudent en manipulation
  • Antivol : investissez dans un bon antivol, les VAE sont prisés des voleurs

Erreurs courantes à éviter

  • Mode turbo permanent : vous viderez la batterie trop vite
  • Oublier de pédaler : le VAE aide, mais il faut pédaler
  • Cadence trop faible : pédalez à 60-80 tours/min pour l'efficacité
  • Assistance en descente : coupez l'assistance, économisez la batterie
  • Batterie vide stockée : risque d'endommagement définitif
  • Nettoyage haute pression : peut endommager l'électronique

Pour aller plus loin

  • Autonomie et dénivelé
  • Nettoyer la chaîne
  • Louer un VAE
