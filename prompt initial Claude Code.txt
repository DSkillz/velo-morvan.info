SCRIPT PYTHON OBLIGATOIRE — SYNCHRO ASSETS AMAZON (OUTIL)
- Créer un dossier outils :
  /tools
    └── amazon_assets_sync.py

Objectif du script :
- Lire /data/amazon.json
- Pour chaque produit :
  - Construire affiliate_url : https://www.amazon.fr/dp/{asin}/?tag=jack59-21
  - Récupérer l’image principale du produit via une source AMAZON avec scraping le plus efficace.
    (Le script est un outil offline, donc les appels externes sont autorisés côté outil.)
  - Télécharger l’image en local
  - Convertir en WEBP qualité 85%
  - Écrire dans /assets/img/products/{asin}.webp
- Générer /data/amazon.enriched.json contenant :
  { slug, title, asin, affiliate_url, local_image, usage, tags }

Contraintes du script :
- Autorisation : scraping de pages Amazon HTML.
- Le script doit :
  - gérer les erreurs proprement (passer au produit suivant de la recherche de produit)
  - conserver un fallback /assets/img/product-placeholder.webp
  - être idempotent (ne retélécharge pas si le fichier existe, sauf flag --force)
- Le script doit fournir une GUI avec logs détaillés au maximum :
  - python tools/amazon_assets_sync.py
- Dépendances Python :
  - installation automatique des dépendances
  - utiliser requests + pillow (PIL) pour conversion webp
  - fournir requirements.txt dans /tools/requirements.txt
- Documenter dans README :
  - installation : pip install -r tools/requirements.txt
  - comment lancer le script
  - rappel : le site final ne fait aucun appel externe (hors GA4).
